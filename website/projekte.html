Cloud / IT Projekte

Cloud Resume

Ich habe ein Cloud-Resume-Projekt erstellt, das moderne AWS-Technologien nutzt, um eine vollständig gehostete, dynamische Webseite bereitzustellen. Dafür habe ich HTML und CSS verwendet, um das Resume als statische Webseite zu gestalten und mit Amazon S3 als statische Website zu hosten. Um HTTPS zu ermöglichen, habe ich CloudFront genutzt und mit Route 53 eine eigene Domain eingebunden. Ein Besucherzähler wurde mit JavaScript implementiert, wobei die Zugriffszahlen in DynamoDB gespeichert werden. Die Kommunikation zwischen Frontend und Datenbank erfolgt über eine API, die mit API Gateway und Lambda (Python, boto3) umgesetzt wurde. Die gesamte Infrastruktur wurde mit AWS SAM (Infrastructure as Code) definiert und über GitHub Actions (CI/CD) automatisiert bereitgestellt. Das Frontend-Repo sorgt für Updates des S3-Buckets, während das Backend-Repo Änderungen an der API testet und ausrollt.

View Counter

Ich habe den View Counter mit JavaScript implementiert, der über eine API Gateway-Anfrage eine AWS Lambda-Funktion aufruft, die den Besucherzähler in einer DynamoDB-Tabelle speichert und aktualisiert.

Finanz-Newsletter

Ich habe eine automatisierte Finanz-News-Webseite entwickelt, die täglich große Marktveränderungen und anstehende Events aus verschiedenen Quellen aggregiert. Dazu nutze ich AWS Lambda für die Datenverarbeitung, Amazon DynamoDB zur Speicherung, sowie API Gateway für dynamische Abrufe. Die Daten stammen aus Finanz-APIs, Web-Scraping und der Twitter API. Die Webseite wird mit React und AWS Amplify oder S3 + CloudFront gehostet. Zusätzlich sorgen EventBridge für automatisierte Updates und Amazon Comprehend für Sentiment-Analysen. Damit habe ich eine skalierbare und cloudbasierte Lösung geschaffen, die Finanzdaten effizient aufbereitet und visualisiert.
